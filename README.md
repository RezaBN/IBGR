# IBGR
IBGR (Influence-Based Group Recommendation) is a novel group recommendation that published in Knowledge-Based System journal at Elsevier. It takes into account the influence of members and leaders in groups to determine items rating proper to all members for groups. There is a sample MATLAB code of IBGR for a fixed group with 5 member and 7 items.

Paper is avalable in the following link: https://www.sciencedirect.com/science/article/pii/S0950705120304706
